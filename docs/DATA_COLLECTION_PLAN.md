## 데이터 품질 전략

### 1. 신뢰도 계층화
- **Tier 1 (공신력)**: IEA, BloombergNEF, LGES 사업보고서, 감사 리포트
- **Tier 2 (분석)**: 언론사 기사 (경제지, 영문 뉴스)
- **Tier 3 (기술)**: 학술 논문 (peer-reviewed journals, arXiv)

### 2. 확증 편향 방지
- **다양한 관점**: LGES(Tier 1) vs CATL(Tier 2) 자료 비중 차이 → Web Search로 보정
- **기사 선정**: 국내(동아·경향·머니투데이) + 국제(Bloomberg·36Kr·ESS News) 혼합
- **학술 자료**: 중국·미국·한국 저자 논문 포함

### 3. 시간성
- 2024~2026년 최신 자료 우선 (시장 변화 민감한 분야)
- LGES 2025 재무 공시, CATL Q2 실적 포함
